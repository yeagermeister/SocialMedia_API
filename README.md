# SocialMediaAPI
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)     
This is the backend for a generic social media site.  It has routes for user functions, to include showing friendships.  It has the ability to post  and update thoughts, and add reactions to those thoughts.
